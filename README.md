# Safety & Connectivity App

A hackathon project built with **React** (frontend) and **Node.js** (backend) that integrates with **ESP32 LoRa mesh** for real-time communication and tracking.  
This project is designed to enhance **safety, accessibility, and connectivity** with smart features like live location, SOS alerts, and emergency contacts.

---

## 🚀 Features
- 🌍 **Real-time Location Tracking** – Displays live location using Google Maps API with latitude & longitude.  
- 🆘 **SOS Alerts** – 5 different SOS buttons to send safety signals quickly.  
- 📞 **Emergency Contacts** – View, add, and manage important emergency numbers.  
- 🌐 **Live Translator** – Integrated translation for better communication.  
- 📡 **LoRa Mesh + ESP32 Integration** – Fetches and transmits data efficiently over mesh networks.  
- 🙋 **About Me Section** – Personalized space to share details.  

---

## 🛠️ Tech Stack
- **Frontend**: React  
- **Backend**: Node.js, Express  
- **Other Tools**: Nodemon, CORS  
- **APIs/Hardware**: Google Maps API, ESP32, LoRa Mesh
